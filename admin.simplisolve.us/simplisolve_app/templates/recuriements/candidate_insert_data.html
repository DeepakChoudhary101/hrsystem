  • 01 Data Scientist
    Key Responsibilities:
    • Make thoughtful decisions around which methods/algorithms are likely to work well in solving the business problem at hand.
    • Consider the quality of the available input data and build robust systems that will work well in the presence of noise/errors.
    • Measure the model output in practical real-world settings and iterate your approach.
    Requirements:
    • Experience in LLM and Generative AI
    • Experience in Document extraction/chat
    • You are experienced with SQL, handling large-scale data, and are comfortable with at least one programming language (Python, R, etc.).
    • Experience working in a cloud-native environment such as AWS
    • Experience supporting ML models development on big data infrastructure (on knowledge graph would be a bonus)
    • You have experience building ML models using modern ML approaches like Neural Nets or Tree-ensembles from scratch for new applications - making decisions relating to which supervised labels to use, the metric to optimize for, and the features likely to be useful.
  • 02 Machine Learning Engineer
    Key Responsibilities:
    • Data Engineering: Collect, preprocess, and clean data to make it suitable for machine learning tasks. This may involve data wrangling, feature engineering, and data augmentation.
    • Feature Selection: Identify relevant features and optimize them for model performance and efficiency.
    • Model Evaluation: Assess model performance using appropriate metrics, conduct cross-validation, and fine-tune hyperparameters to improve model accuracy and generalization.
    • Model Deployment: Deploy machine learning models into production environments, often using containerization (e.g., Docker) and orchestration tools (e.g., Kubernetes).
    • Monitoring and Maintenance: Implement monitoring and logging systems to track model performance, and proactively address issues that may arise in production.
    Requirements:
    • Programming Skills: Proficiency in programming languages such as Python or R, as well as experience with relevant libraries and frameworks.
    • Machine Learning Knowledge: Strong understanding of machine learning algorithms, techniques, and concepts.
    • Relationships. Ability to build and effectively manage relationships with business leaders and external constituents
    • Experience working in a cloud-native environment such as AWS
    • Culture. Excellent judgement, impeccable ethics, and a strong team player; desire to succeed and grow in a fast-paced, demanding and entrepreneurial Company.
    • Strong communication skills (developing performant robotic solutions requires cross-team communication/collaboration)
  • 03 Software Application Developer

    • Ensures software standards are met. Analyzes user needs and software requirements to determine feasibility of design within time and cost constraints.
    • Troubleshoots production problems related to software applications.
    • Initiate action for evaluation and recommendation of application software packages, application integration, and testing tools.
    • Implement recommendations for improving documentation and development process standards
    Requirements:
    • Programming Skills: Proficiency in programming languages such as Python or R, as well as experience with relevant libraries and frameworks.
    • Experience with two or more of the following: SQL, Silk, JSON, Data Analytics, MongoDB, Elastic Search, Postgres, Kubernetes, Docker, AWS, View Jazz Framework, Process Automation, UI Development, CI/CD knowledge

    • Experience working in a cloud-native environment such as AWS
  • 04 Research Scientist
    Key Responsibilities:
    • Use your deep understanding of machine learning to tackle meaningful technical problems.
    • Collaborate with product development teams to transfer your ideas into product solutions.
    • Perform fundamental research by defining, designing, implementing, and evaluating algorithms.
    • Actively engage with the academic community by collaborating with universities, publishing and presenting your work, and attending conferences.
    • Be results-oriented in the face of ambiguous problems and uncertain outcomes.
    Requirements:
    • Proven experience as a machine learning researcher with research publications in the model compression (quantization/pruning) or generative AI / NLG / LLMs space.
    • Strong background in deep learning with expertise in one or more of computer vision, NLP, speech, reinforcement learning, generative models, etc.
    • Knowledge of common ML frameworks (like PyTorch or Keras) and libraries (like NumPy and scikit-learn)
    • Strong research background, as demonstrated through papers and a graduate degree (MS or PhD) in Electrical Engineering, Computer Sciences, Mathematics, or equivalent experience
    • Ability to explain and present analysis and machine learning concepts to a broad technical audience.
